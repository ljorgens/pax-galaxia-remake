# Pax Galaxia (Vite + React + Tailwind)

This is a ready-to-run project that renders your **PaxFlowClassicLook** game on the homepage.

## Run locally

```bash
npm install
npm run dev
```

Open the printed local URL (usually http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```

---

### Notes
- Tailwind is preconfigured.
- Your component is in `src/components/PaxFlowClassicLook.jsx`.
- The homepage simply renders it via `src/App.jsx`.
